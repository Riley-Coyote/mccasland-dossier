# The McCasland Dossier

An open-source investigative archive documenting the disappearance of Air Force Maj. Gen. William Neil McCasland (Ret.) and the institutional context surrounding it.

## Live Site

**[View the dossier →](https://riley-coyote.github.io/mccasland-dossier/)**

## Structure

- **Home** — Editorial landing page with subject profile and guided reading order
- **Analytical Overview** — Convergence map, evidence framework, route descriptions
- **UAP / SAP Nexus** — SAPOC oversight architecture and McCasland's role within it
- **Disclosure Network** — The Podesta emails, DeLonge, TTSA, and the advisory team
- **Wright-Patterson Chain** — Institutional lineage from 1947 Wright Field to AFRL
- **Kirtland / NM Nexus** — Geographic density of the disappearance location
- **Disappearance Forensics** — Timeline, Silver Alert contradiction, scenario analysis

## Evidence Discipline

Every claim is tagged with one of four evidence labels:

- 🟢 **Documented** — On the public record
- 🟡 **Claimed** — Stated by a named source, not independently verified
- 🔵 **Inferred** — Analytical conclusion drawn from documented facts
- 🔴 **Disputed** — Actively contested between credible sources

## Source Material

Built from public-domain sources, WikiLeaks archives, official Air Force records, and open-source reporting. The original research markdown files are included in the repository.
